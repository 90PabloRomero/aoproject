Comandos

Comando Descripción
-TU MENSAJE Sirve para mandar un mensaje en color rojo dentro del juego
/UPTIME Muestra el tiempo en el que el servidor se mantuvo online
/ONLINE Muestra la cantidad de usuarios online
/PING Muestra el ping actual en MiliSegundos (1000 ms = 1 seg)
/DESC Sirve para cambiar la descripción de tu personaje
/EST Esto les muestra el estado del personaje. Les indica la cantidad de vida, maná y energía que tiene, además de indicarles en qué mapa y en qué coordenadas se encuentra. También indica el mayor y menor golpe, y si tiene un arma equipada, mostrará en paréntesis sus atributos; lo mismo sucede con la defensa.
\NICK Con este comando mandarás mensajes privados a un jugador que esté en la pantalla del juego. Es importante destacar que utiliza la barra invertida a diferencia de los otros comandos. Si el nombre es compuesto (Ej: Aurora Plus) se deberá utilizar el + en el lugar del espacio. (Ej: \AURORA+PLUS HOLA)
/PENAS NICK Con este comando verás las penas asignadas a el personaje escrito
/SALIR Sirve para salir del juego a la pantalla principal (Conectar)
/gm Sirve para llamar a un Game Master o enviar una consulta
/denunciar TU MENSAJE Sirve para realizar una denuncia dentro del juego ya sea por comportamiento indebido o algún hecho ilícito (Ej: explotación de bugs)
Importante: El mal uso de este comando está penado
/ROL Tu mensaje Realizarás una consulta a el staff de Role Masters o bien éste podrá ser el canal de comunicación para algún evento.
/COMERCIAR Sirve para comerciar con un usuario o un NPC, en el caso del NPC también podrás darle doble click en los pies para comerciar.
/APOSTAR X Sirve para apostar con los NPCS Apostadores (X=Cantidad de oro que desea apostar)
/MEDITAR (o F6) El personaje que tenga maná, comenzará a meditar para recuperarla.
/DESCANSAR Sirve para recuperar energía junto a una fogata
/RESUCITAR Sirve para resucitarte con el sacerdote de cada ciudad
/CURAR Sirve para que el sacerdote te cure
/BOVEDA Sirve para interactuar con el banquero
/BALANCE Verás el dinero que tiene en el banco el personaje
/RETIRAR X Retira X cantidad de oro del banco
/DEPOSITAR X Deposita X cantidad de oro en el banco
/ENLISTAR Cliqueando a Fressenda o el Demonio, servirá para enlistarte en sus filas
/RECOMPENSA Para solicitar la recompensa de acuerdo a los objetivos faccionarios (sobre el Demonio o Fressenda, la regente de Hauteville)
/RETIRARFACCION Sirve para retirarse de la facción en la que estás enlistado/a
/FUNDARCLAN Sirve para fundar un clan
/ONLINECLAN Sirve para ver los usuarios online dentro del Clan
/CMSG Sirve para comunicarse con los miembros del clan que estén conectados
/VOTO NICK Sirve para votar al nuevo líder del clan, en el caso de estén las inscripcciones abiertas
/SALIRCLAN Sirve para salir del clan
/QUIETO Sirve para que la mascota se quede quieta en el lugar. Se debe primero clickear la mascota y luego tipear el comando.
/ACOMPAÑAR Sirve para que la mascota te siga. Se debe primero clickear sobre ella.
/LIBERAR Liberarás la mascota y perderás el control sobre ella
/COMPARTIRNPC Permite compartir los npcs con usuarios del mismo status sin estar en clan/party
/CREARPARTY Crea una party
/PARTY Solicita unirse a la party
/ACCEPTPARTY NICK Acepta al usuario a la party
/ONLINEPARTY Verás los usuarios conectados en party
/PMSG Mandarás un mensaje a la party
/SALIRPARTY Saldrás de la party
/ECHARPARTY NICK Echarás al usuario de la party
/PARTYLIDER NICK Sirve para pasar el liderazgo de la party a otro user
/HOGAR Sirve para volver a tu ciudad natal sin tener que moverse
/INFOQUEST Sirve para ver la información de la quest en curso
/QUEST Sirve para ver las opciones de quest y aceptar la misma
